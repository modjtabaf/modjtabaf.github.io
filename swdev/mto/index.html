Materials Take-Off (MTO) Software

MTO software was designed as a post-processor for PV Elite software. It was developed by a team of three engineers. I was the sole developer of the software. I also derived and implemented an algorithm for optimal arrangement of small plates resulted from vessel parts breakdown over user-defined sheet metals. The first step in using the software is to import the final report of PV Elite software. Several reports can be imported and merged into a single project. The user can define the list of available sheet metal plate sizes for each material. The software can handle rings and cones by breaking them down to several pieces. It can generate the list of parts and materials for each vessel, estimate theÂ cost for each vessel and the whole project, generate MTO table for the whole project, and generate optimal cutting plan for the whole project with graphical representation. The generated optimal cutting plan can be exported as an AutoCAD script. It also allows the user to define custom vessels. User-friendly environment of the software allows the user to easily specify all parameters, perform analysis, and view the results. All features are classification into various categories for easier access. Plate Arrangement module is equipped with a 3D graphical overview. Open and Save features are provided as well. The software was developed using C++ programming language under Microsoft Window OS. 3D views were generated using OpenGL library. GDI+ was used for 2D graphics, e.g. plate arrangement results.
